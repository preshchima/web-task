HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define your page content, structure and meaning.
You don't use it for styling purposes. Use CSS for that instead.

  • HTMl uses 'elements' to describe(annotate) content
  • HTML elements typically have an opening tag and a closing tag
  • you can also have void elements like images
  • you can also configure elements with attribute
  • There is a long list of available elements but you will gain experience over time

learn more about all avaailable html elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Style Sheets) is a used for styling your page content.

  • Styles are assigned via property-value pair
  • You can assign style via the "style" attribute
  • To avoid code duplication, you typocally use lobal styles(e.g via the "style" element) instead
  • Alternatively, you can work with external stylesheet files which you 'link' to
  • When working with CSS, concepts like "inheriatnce", "specificity" and "Box model" should be understood

learn more about all avaailable html elements on the MDN CSS property reference.
